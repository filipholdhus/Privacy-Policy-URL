Personvernerklaring for KollektivAppen

Sist oppdatert: 10. mai 2026

1. Kven er ansvarleg?

KollektivAppen blir utvikla og drifta av Filip Holdhus. Ved spørsmål om personvern kan du kontakte oss på filip.holdhus@gmail.com.

2. Kva data samlar appen inn?

Appen kan lagre desse opplysningane når du brukar tenesta:

  • Namn og telefonnummer som du sjølv skriv inn.
  • Profilbilde dersom du vel å laste opp dette.
  • Kollektiv, invitasjonskode og medlemskap i kollektiv.
  • Meldingar, bookingar, handleliste, kjøleskapsvarer, kalenderhendingar og oppgåver du opprettar.
  • Oppgjør/utgifter, inkludert beløp, kven som la ut, og kven som har markert betalt.
  • Push-varslings-token dersom du tillèt varslingar.
  • Teknisk informasjon som er nødvendig for sikker drift av Firebase og Expo.

3. Kvifor brukar vi data?

Data blir brukt for å levere funksjonane i appen:

  • opprette og administrere kollektiv
  • vise relevant innhald til medlemmar i same kollektiv
  • handtere booking, handleliste, kalender og oppgjør
  • sende push-varsel når du har samtykka til det
  • beskytte appen mot misbruk og uautorisert tilgang

4. Kven blir data delt med?

Data blir ikkje seld. Appen brukar tredjepartstenester som er nødvendige for drift:

  • Firebase/Google Cloud for innlogging, database og lagring.
  • Expo for bygging, push-varslingar og TestFlight-distribusjon.
  • Apple for TestFlight og App Store-distribusjon.

Medlemmar i same kollektiv kan sjå informasjon som er nødvendig for felles funksjonar, til dømes namn, telefonnummer, oppgjør og innlegg.

5. Lagring og sletting

Data blir lagra så lenge brukarkontoen eller kollektivet er aktivt, eller så lenge det er nødvendig for å levere tenesta. Du kan be om innsyn, eksport eller sletting av data. Appen har funksjon for kontosletting, der personopplysningar blir sletta eller anonymiserte der det er nødvendig for å halde felles historikk og oppgjør forståeleg.

6. Dine rettar

Du kan be om:

  • innsyn i data vi har om deg
  • retting av feil informasjon
  • eksport av eigne data
  • sletting av konto og personopplysningar
  • tilbaketrekking av samtykke til push-varsel

Kontakt oss på filip.holdhus@gmail.com dersom du vil bruke desse rettane.

7. Born

Appen er laga for studentar og vaksne i kollektiv. Tenesta er ikkje retta mot born under 13 år.

8. Endringar

Vi kan oppdatere denne personvernerkleringa dersom appen eller databehandlinga endrar seg. Nyaste versjon vil alltid ligge på denne sida.
